# 🏃 Get up and Run!

A CLI command that automates the setup of various terminal tabs with different commands based on a `get_up_and_run.json` file

![demo](./demo.gif)

## 📦 Install

`npm install -g get-up-and-run`
`yarn add global get-up-and-run`

## Usage

`run <commandsuite>`
